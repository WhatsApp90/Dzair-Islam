# 🇩🇿 البوت الإسلامي الجزائري التفاعلي (Dz Islamic Bot)

بوت تلغرام إسلامي وشامل بالدارجة الجزائرية، يوفر مواقيت الصلاة لجميع ولايات الجزائر الـ 48، استماع للقرآن بأصوات كبار القراء الجزائريين، مسابقات تفاعلية، والبحث عن المساجد والأماكن القريبة.

---

## ✨ المميزات الرئيسية

* 🕌 **مواقيت الصلاة والتنبيهات:** دعم كامل لجميع ولايات الجزائر الـ 48 مع أذان وتنبيهات أوتوماتيكية.
* 📖 **القرآن الكريم:** الاستماع للتلاوات بأصوات أشهر القراء الجزائريين (مثل الشيخ ياسين الجزائري، سعيد دباح، وغيرهم).
* 📍 **خدمة الخرائط:** إمكانية إرسال الموقع لمعرفة أقرب مسجد أو محل حلال أو مقهى.
* 🧠 **مسابقة إسلامية تفاعلية:** أسئلة دينية وثقافية بالدارجة مع حاسبة نقاط وترتيب أسبوعي للأوائل كل خميس.
* 💡 **محتوى إيماني وثقافي يومي:** أذكار الصباح والمساء، حكم شعبية جزائرية، ومعلومات قرآنية متجددة.
* 📢 **نظام إشعار التحديثات:** إرسال رسالة آلية عند بدء التشغيل بعد التحديث للتأكد من وصول البوت لجميع المشتركين دون فقدانهم.

---
